# Draw binary indicators given true class memberships

Draw binary indicators given true class memberships

## Usage

``` r
draw_indicators(X, rho)
```

## Arguments

- X:

  Integer vector of length n. True latent class (1-indexed).

- rho:

  T x K matrix. `rho[t, k] = P(Y_k = 1 | X = t)`.

## Value

An n x K integer matrix of 0/1 values.
